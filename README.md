# Netty使用及推荐学习路线

#### 介绍
netty是dubbo,elasticsearch,以及众多消息中间件的底层通信框架 (课程总计：53小时)

#### 版本 V1.0

#### 软件架构
服务器端
1. 自定义通讯协议，编解码器
2. 解决服务器端内存泄漏的情况
3. 解决TCP拆包粘包的问题
4. QPS 10万
5. 设置高水位值
    
客户端
1. 自定义通讯协议，编解码器
2. 解决TCP拆包粘包的问题
3. 定时发送心跳
4. 解决客户端出现长时间阻塞时，依旧可以正常运行（非必要，不建议使用）
5. 断线重连功能

#### 使用说明

1.  设备号为16位，如需更改，更改编解码器即可。
2.  运行客户端，服务器端即可。
